# Руководство по развертыванию на Render.com

## 1. Подготовка файлов
Создайте проект со следующей структурой:

osint-dossier/
├── main.py
├── web.html
├── web.css
├── web.js
├── requirements.txt
└── render.yaml

## 2. Создание репозитория на GitHub
1. Создайте новый репозиторий на GitHub
2. Загрузите все файлы проекта
3. Убедитесь, что requirements.txt и render.yaml находятся в корне

## 3. Развертывание на Render.com
1. Зайдите на https://render.com
2. Нажмите "New +" → "Web Service"
3. Подключите ваш GitHub репозиторий
4. Укажите настройки:
   - Name: osint-dossier
   - Environment: Python
   - Build Command: pip install -r requirements.txt
   - Start Command: python main.py
5. Нажмите "Create Web Service"

## 4. После развертывания
- Сайт будет доступен по адресу: https://osint-dossier.onrender.com
- Первый запуск может занять 2-5 минут
- Бесплатный план имеет ограничения:
  * 750 часов работы в месяц
  * Авто-отключение после 15 минут неактивности
  * Перезапуск при следующем запросе

## 5. Проверка работоспособности
После развертывания проверьте:
1. Открывается ли главная страница
2. Работает ли перетаскивание блоков
3. Сохраняются ли изображения
4. Работает ли мобильная версия

## Примечания
- На Render.com папка 'icon' не требуется, так как иконки генерируются кодом
- Для продакшена рекомендуется использовать платный план
- Для увеличения производительности можно добавить gunicorn
